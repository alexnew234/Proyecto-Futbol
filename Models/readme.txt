ARQUITECTURA DEL MODELO DE DATOS

En cumplimiento con el requisito del proyecto:
"Utiliza el módulo de acceso a la base de datos [...] como una librería local instalable. 
No lo incluyas como un módulo en el proyecto."

La lógica de base de datos (el Modelo) no se encuentra en este directorio, 
sino que ha sido extraída a la librería externa 'torneo_db'.

Para revisar el código del modelo:
1. Ver la carpeta adjunta 'torneofutbol_db'.
2. La base de datos física se genera en la ruta de usuario del sistema 
   (C:\Users\[Usuario]\TorneoFutbolData) para asegurar la persistencia.
